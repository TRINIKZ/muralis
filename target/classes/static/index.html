PROJECT 01 - MURALIS
Clientes Gerador

SISTEMA DE MANIPULAÇÃO DE DADOS

O sistema consiste em uma aplicação web para gestão de clientes e seus contatos, desenvolvida como solução para um desafio técnico. Seu objetivo é substituir o uso de agendas físicas por uma plataforma digital, permitindo o cadastro, edição, exclusão e consulta de informações de forma organizada e eficiente.

A aplicação possibilita o gerenciamento completo de clientes, incluindo busca por nome ou CPF, além do controle de múltiplos contatos vinculados a cada cliente, como telefone e e-mail. Dessa forma, centraliza as informações e facilita o acesso e a manutenção dos dados.

O projeto foi desenvolvido de forma independente com o intuito de demonstrar habilidades em análise de requisitos, modelagem de dados e desenvolvimento fullstack, utilizando Java com Spring Boot no backend e tecnologias web no frontend, seguindo boas práticas de organização e manutenção de código.

Além de atender aos requisitos propostos, o sistema representa uma solução prática para um problema comum no ambiente corporativo, oferecendo uma alternativa mais segura, estruturada e eficiente para o gerenciamento de informações de clientes.

Ferramentas de inteligência artificial foram utilizadas como apoio no desenvolvimento, auxiliando na geração de código e na resolução de dúvidas técnicas, mantendo-se, porém, total responsabilidade sobre as decisões de implementação e o resultado final do sistema.
